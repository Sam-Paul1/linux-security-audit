# Linux Security Audit

A Bash script that performs a quick Linux security audit.

## Features

- System information
- Network information
- Firewall status
- Listening ports
- Running processes
- Logged-in users
- RAM usage
- Disk usage
- SSH status
- Summary report

## Usage

```bash
chmod +x security.sh
./security.sh
```

## Technologies

- Bash
- awk
- grep
- cut
- systemctl
- ss
- free
- df
